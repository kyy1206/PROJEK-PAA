# Modul Dzaky Ribal Faiz

## Tanggung Jawab
Modul ini berfokus pada konstruksi peta dan jalan.

## Bagian yang Dikerjakan
- Membuat struktur jalan pada peta.
- Membuat jalan luar/ring road.
- Membuat koneksi jalan dalam.
- Mengubah data jalan menjadi bentuk vector node dan edge.
- Menjaga jalan agar tetap berada di area peta.

## File Terkait
- `map/grid/roads.py`
- `map/grid/grid_kota.py`
- `map/grid/vector.py`
- `map/grid/clip.py`
- `map/utils.py`

## Catatan
File pada modul ini merupakan bagian kontribusi anggota. Program final hasil integrasi berada pada folder `00_RUNTIME_PROGRAM`.
